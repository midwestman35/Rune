This template should help get you started# Rune V4.1

Rune is a specialized log analysis tool designed for telephony and VoIP session logs.

## Features (v4.1)

-   **Dynamic File Loading**: Open any `.log` file to analyze it instantly.
-   **Timeline Visualization**: "Scrubber" view to spot errors across the entire timeline.
-   **Smart Dashboards**: Automated charts for Error/Warning distribution and Log Volume.
-   **Live Tail**: (Prototype) Real-time monitoring view.
